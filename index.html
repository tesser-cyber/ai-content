3. Trading Calculations & Formulas

Margin Calculation (Forex):
Margin = (Lot * Contract Size / Leverage) * Exchange Rate [cite: 945]

Pip Value Formula:
Pip Value = ((Contract Size * Lot) * 10^-digits) / Exchange Rate [cite: 865]

Swap Rate Formula:
Swap = Pip Value * Swap Rate * Number of Holding Days [cite: 1094]
