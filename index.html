Laos Deposit Process (ฝากเงินลาว)

ขั้นตอนการฝากเงินสำหรับลูกค้าในประเทศลาว (LAK - Kip)

How to deposit in Laos?

  1. กดปุ่ม Deposit ในพื้นที่สมาชิก
  2. เลือกบัญชีเทรดที่ต้องการฝากเงิน
  3. ระบุจำนวนเงินที่ต้องการฝาก
Note:
  • ยอดฝากขั้นต่ำคือ $50
  • ระบบจะคำนวณอัตราแลกเปลี่ยนเป็นเงิน LAK (Kip) ให้โดยอัตโนมัติ

After completing the transfer

ต้องแจ้งข้อมูลต่อไปนี้ในระบบหลังจากโอนเสร็จ:

  • Sender’s bank account number
  • Transfer date (based on the payment slip)
  • Upload the transfer slip
Important: เลขบัญชีและวันที่โอนต้องตรงกับในสลิป มิฉะนั้นการฝากอาจล่าช้าหรือถูกปฏิเสธ

3. Formulas & Technical Calculations

Forex/Gold Margin Calculation [cite: 943]

Margin = (Lot * Contract Size / Leverage) * Exchange Rate [cite: 945]

Pip Value Formula [cite: 865]

Pip Value = ((Contract Size * Lot) * 10^-digits) / Exchange Rate [cite: 865, 1095]

Swap Rate Formula [cite: 1094]

Swap = Pip Value * Swap Rate * Number of Holding Days [cite: 1094]
